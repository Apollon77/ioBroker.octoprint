## State text to State Id

### STATE_NONE

- Offline

### STATE_OPEN_SERIAL

- Opening serial connection

### STATE_DETECT_SERIAL

- Detecting serial connection

### STATE_CONNECTING

- Connecting

### STATE_OPERATIONAL

- Operational

### STATE_STARTING

- Starting
- Starting print from SD
- Starting to send file to SD

### STATE_PRINTING

- Printing
- Printing from SD
- Sending file to SD

### STATE_CANCELLING

- Cancelling

### STATE_PAUSING

- Pausing

### STATE_PAUSED

- Paused

### STATE_RESUMING

- Resuming

### STATE_FINISHING

- Finishing

### STATE_CLOSED

- Closed

### STATE_ERROR

- Error

### STATE_CLOSED_WITH_ERROR

- Offline after error

### STATE_TRANSFERING_FILE

- Transferring file to SD
